# TestWithHedgehogs-TestTaskForAdmixer-
The implementation of calculating the minimum number of steps contains interfaces from which you can implement 
classes for input and output of information, a class with a search algorithm. A hedgehog model was also added, 
with the intention of further extending the code and functionality, but the class itself only uses a static variable that 
specifies the number of hedgehog colors.

Реалізація вирахування мінімальної кількості кроків містить інтерфейси, від яких можна реалізовувати класи вводу 
та виводу інформації, клас з алгоритмом пошуку. Також була додана модель їжака, з розрахунком подальшого розширення 
коду та функціональності, але у самому класі використовується лише статична змінна, яка задає кількість кольорів їжаків.
